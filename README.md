# 🚗 AI-Powered Car Rental System with Chatbot and Automated Damage Detection

## Overview

This project presents an AI-powered car rental platform designed to modernize traditional rental systems. It integrates artificial intelligence to automate customer interaction and vehicle inspection processes. The system includes an AI chatbot to assist users with bookings and queries, and a computer vision module to detect vehicle damage automatically from images.

## Features

* AI Chatbot for answering customer queries and assisting with bookings
* Car booking and rental management system
* Automated vehicle damage detection using computer vision
* User authentication and profile management
* Vehicle inventory management
* Rental history tracking

## Technologies Used

* Frontend: HTML, CSS, JavaScript
* Backend: Node.js / Python
* AI/ML: TensorFlow, YOLO, CNN
* Database: MongoDB / MySQL
* Version Control: Git & GitHub

## System Modules

1. User Registration and Login
2. Car Listing and Booking
3. AI Chatbot Support
4. Vehicle Damage Detection
5. Admin Dashboard

## How It Works

1. Users register and log into the platform.
2. They can browse available cars and make bookings.
3. The AI chatbot assists users with queries and booking guidance.
4. Vehicle images are analyzed by the AI model to detect damages.
5. Admins can manage vehicles, bookings, and damage reports.
